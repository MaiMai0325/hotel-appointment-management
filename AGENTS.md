# AGENTS.md

このプロジェクトは森下ホテルの民宿運営専用予約管理システムです。

目的：
Airbnb、Booking.com、Expedia、携程、直接予約などを一元管理する。

重要ルール：
- 既存機能を壊さない
- 既存デザインを大きく変えない
- 日本語UIを維持する
- スマホで使いやすくする
- Excel運用を壊さない
- チェックアウト日は空室扱いにする
- まずは localStorage 保存で進める
- 大きな変更前には計画を出す
- 変更後は変更ファイルと確認方法を説明する

優先機能：
1. 予約データ構造の整理
2. 予約一覧の検索・絞り込み・並び替え
3. 今日のチェックイン/アウト/清掃表示の強化
4. 予約元別売上集計
5. 支払い未了・清掃未了の管理
6. Excel連携の安全整理
